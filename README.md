
# Vue 電商網站 - Panda nails美甲

Demo: https://yeif.github.io/panda_nails-layout/#/

使用Vue-Cli 當開發環境，並使用Option API做開發
## 前台：

* 首頁
* 產品頁面
* 訂單查詢
* 常見問題 
* 部落格
* 收藏頁面 (我的最愛)
* 購物車 
## 後台

* 新增/編輯 產品頁面
* 新增/編輯 優惠卷頁面
* 新增/編輯部落格頁面
* 訂單頁面


## 使用技能：

* Vue Cli 環境建立
* Vue Router 路由配置
* 使用 JacaScript ES6
* 搭配 Bootstrap5 完成 RWD 切版


* 使用第三方套件 Vee Validate 製作表單驗證
* 使用 Axios 套件取得 RESTful API 資料 
* 使用 swiper 套件 做輪播效果
* 使用 AOS 套件做視差滾動
